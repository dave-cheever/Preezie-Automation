{{ResponseLLM}}

TASK:

1. Review the LLMRequestFormattedPrompt to understand what intent categories are available.
2. Review PromptArguments to understand:
   - userPrompt: the user's original message
   - brandOverview: context about the brand/store
   - choices: the available intent options
3. Analyze the UserMessage to understand what the user is trying to accomplish.
4. Determine whether the Intent in AssistantResponse is:
   - Correctly classified based on the user's message
   - Appropriate for the context given in PromptArguments
   - Consistent with the available choices
5. Judge correctness of classification, not exact wording.

SCORING RULES (0–5):
- relevance: the intent correctly addresses what the user wants
- faithfulness: no misclassification or invented intents
- instructionCompliance: follows the intent categories defined in the prompt
- semanticCloseness: the classified intent accurately represents user's intention

PASS CONDITIONS:
- relevance >= 4
- faithfulness >= 4
- instructionCompliance >= 4
- semanticCloseness >= 4

RETURN ONLY THIS JSON STRUCTURE:

  },
